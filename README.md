Unlike traditional "input-output" ML models, KrishiCare operates as an Autonomous Agent:
Perception: It "sees" the plant via Computer Vision.

Feedback Loop: Through the chat interface, it maintains a continuous state of assistance, adapting its advice as the farmer provides more details.
